# Data Science Portfolio

Personal portfolio website highlighting achievements and updates.

## Tech Stack
- **Frontend:** React + Vite
- **Design:** Created with Claude AI assistance
- **Deployment:** Github Pages

## Sections
- About: Background in Data Science & Physics at NJIT
- Projects: Featured research and ML applications
- News: Recent professional updates
- Contact: Links to GitHub, LinkedIn, and email

## Development
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
